## Bugs Fixed

### Bug 1: JS `slice` method flagged as field contract violation (field_contract.py)
**Severity**: Minor (test failure)
**Root Cause**: The `field_name` skip list at `field_contract.py:530-542` did not include common JavaScript built-in Array/String methods. When the frontend code in `Dashboard.jsx:423` called `history.slice(...)` (a standard JS Array method), the field extractor recorded it as accessing field `slice` on object `history`. The validator then looked up `history` in `OBJECT_NAME_MAP`, found it mapped to `fired_alert` (from `/api/alerts/history`), and flagged `slice` as a missing field.

**Fix**: Added 18 common JS built-in methods to the `field_name` skip list in `find_frontend_field_accesses()`:
- Array methods: `slice`, `splice`, `concat`, `join`, `split`, `includes`, `indexOf`, `lastIndexOf`
- String methods: `substring`, `toLowerCase`, `toUpperCase`, `trim`, `replace`, `match`, `charAt`
- Number/formatting: `toFixed`, `toLocaleString`, `getTime`, `getFullYear`, `getMonth`, `getDate`

**Verification**: `test_field_contract_on_real_codebase` now passes (was failing before).

### Bug 2: Missing endpoint mapping for `/api/whale-sentiment/history` (audit_source.py)
**Severity**: Minor (false positive audit finding)
**Root Cause**: The `jsx_endpoint_map` in `audit_source.py:1442` did not include `/api/whale-sentiment/history` for `Dashboard.jsx`, and `known_endpoint_fields` at line 1378 was also missing this endpoint entirely. The frontend calls `/api/whale-sentiment/history` at `Dashboard.jsx:403` to fetch sentiment time-series data, but the audit couldn't find the endpoint's response fields.

**Fix**: 
1. Added `/api/whale-sentiment/history` to `Dashboard.jsx`'s endpoint list in `jsx_endpoint_map`
2. Added `/api/whale-sentiment/history` entry to `known_endpoint_fields` with all response fields: `days`, `history`, `date`, `sentiment_score`, `inflow_usd`, `outflow_usd`, `tx_count`

**Verification**: `d.date` false positive eliminated. Audit minor findings dropped from 1 to 0. Passed checks: 167 → 168.
